# Roadmap

This is a living document. Priorities shift as the project evolves.
OpenFolio is licensed under GNU GPL v3.0 — all shipped features are open source.

---

## Now — v0.1.0 (In Progress)
- [ ] Landing page (Hero, Stats, Features, Why Open Source, How It Works, CTA, Footer)
- [ ] Auth (Login / Signup)
- [ ] Dashboard shell
- [ ] CAMS/CDSL consolidated statement parser

## Next — v0.2.0
- [ ] Stock portfolio tracking (NSE/BSE)
- [ ] Mutual fund tracking with NAV
- [ ] XIRR and CAGR calculations
- [ ] Live price feeds (end-of-day)

## Later — v0.3.0
- [ ] Commodity tracking (Gold, Silver)
- [ ] Portfolio performance charts
- [ ] Tax computation (LTCG/STCG)
- [ ] Mobile responsive dashboard

## Under Consideration
- Zerodha Kite integration (blocked on API access model)
- PWA / mobile app
- Multi-currency support
- Crypto tracking

---

Items in "Under Consideration" have no timeline and may never ship.
Features move up only when implementation is confirmed feasible.
Want to work on something? Open an issue or check [CONTRIBUTING.md](CONTRIBUTING.md).
